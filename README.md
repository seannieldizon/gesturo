# GesTURO — Filipino Sign Language

Learn Filipino Sign Language through interactive lessons designed for both deaf and hearing communities.

## Tech stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Lucide React** (icons)

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Build for production
- `npm run start` — Start production server
- `npm run lint` — Run ESLint

## Pages

- **Home** — Welcome, Why Learn FSL, Quick Start Guide, stats, CTA
- **Learn** — FSL alphabet and common signs & phrases (placeholders)
- **Practice** — Practice and quizzes (placeholder)
- **About** — Mission, accessibility, community

Built with accessibility and inclusivity in mind.
